This script will perform CRUD (Create, Retrieve, Update, Delete) operations on a Google Apps domain using the provisioning API and Google Data Python client library.

If you need to install Python or the Google Data Python Library, the following guide should get you started:
http://code.google.com/apis/gdata/articles/python_client_lib.html
